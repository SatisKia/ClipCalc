Environment setting screen

Environment setting
Press button to display the environment setting screen.
□ [Inv] [Hyp] holds the checked state
When checked, the check state of [Inv] and [Hyp] is retained even if you press etc after pressing [Inv] and [Hyp].
When unchecked, [Inv] and [Hyp] will be unchecked when etc is pressed.
Recalculation
□ When changing the type
If you want the calculation formula to be recalculated when changing the type of calculation, check it.
□ When changing the angle type
Check this box to recalculate the calculation formula when changing angle specification (Deg, Rad, Grad).
□ Before assignment to variable by [STO] button
When checked, the calculation formula is recalculated when the button is pressed, and the calculation result is substituted into the variable memory (The button has the function of button). If not checked, the currently displayed answer will be assigned to the variable memory.
□ Usage is indicated when pressing button
When checked, the usage will be displayed when the button is pressed.
Export profile
When you press the button, the contents of the environment setting are displayed as text. Please select all the text contents, then copy (copy it to the clipboard) and paste it in a notepad etc and save it.
Import profile
Please enter the contents of the environment setting in the text area (copy & paste from Notepad etc.) and press button. The contents entered in the text area do not have to be all the contents at the time of export. Enter only the content you want to reflect.

n-base calculation
● To do n-base calculation, switch to "Signed" type or "Unsigned" type.
● The instructions for n-base are instructions given to the whole computer and methods to give them to individual numerical values.
● For instructions given to the entire computer, you can specify 2 ~ 16 base calculation.
● Binary, octal, decimal, hexadecimal can be specified for instructions given to individual numerical values.
● In n-base calculation, logical operations between numeric values can be performed.
Logical product (&), Logical sum (|), Exclusive OR (^)
● In n-base calculation, numeric bit shift can be performed.
Shift left (<<), Shift right (>>)

Angle type conversion tool
● Press the button with "Floating point", "Fraction", "Mixed fraction", and "Complex" type to start the angle type conversion tool.
● Converts and displays calculated results in degrees, radians, and gradients in real time. The display position of the same value as the calculation result is determined by the current angle specification (Deg, Rad, Grad).
● You can also enter either degree, radians, or gladians directly.

Radix conversion tool
● When you press the button with "Signed" or "Unsigned" type, the radix conversion tool will be activated.
● 2-base, 8-base, 10-base, and 16-base conversion of calculated results in real time and display.
● You can also enter either 2-base, 8-base, 10-base, or 16-base numbers directly.

Time type conversion tool
● By pressing the button in "Time" type, the time type conversion tool will be activated.
● Convert the calculation result to hours, minutes, seconds, frames in real time and display it.
● Either hour, minute, second or frame number can be entered directly.

Program function
● When you press button, the editor starts up.
● There are 26 program memories (a ~ z).
● When the program is changed, button will be displayed. Please press button to save edited contents.
● To call it as a function, use the button. It is input as a function in the calculation formula, and you can enter function parameters subsequently.
● When the button is pressed, edited contents are automatically saved.
● Variable memory (@A ~ @Z) can be referenced by using command ":parent" in the program. For example, when referring to the variable memory @A, write it like ":parent @A".

Technical information
■ Priority of calculation
It is calculated in the following ranking.
(1) !(Factorial)
(2) [~], [-], ^(Power), Function
(3) *, /, %
(4) +, -
(5) <<, >>
(6) &
(7) ^(Exclusive OR)
(8) |
● When function are consecutive, they are executed from the right side to the left side.
exp log sqrt 120 → exp(log(sqrt 120))
● When the operators of the same rank are consecutive, they are executed from the left side to the right side.
12+34-56+78 → ((12+34)-56)+78
● When parentheses are used, parentheses take the highest priority.
■ Input range of the function
Function Input range
asin x -1≦x≦1
acos x -1≦x≦1
acosh x 1≦x
atanh x -1＜x＜1
ln x 0＜x
log x 0＜x
log10 x 0＜x
sqrt x 0≦x
● In case of "Complex" type, there is no limit to the value that can be specified.
